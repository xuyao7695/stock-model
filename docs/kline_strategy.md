# K线形态 + 量能配合交易策略

> **策略定位**：独立于涨停连板战法，基于经典技术分析的K线形态识别 + 量能确认，适用于震荡市和趋势市的波段操作。
>
> **核心原则**：形态是信号，量能是确认，均线是方向，纪律是保障。

---

## 一、策略总览

```
┌─────────────────────────────────────────────────┐
│              K线量能策略选股流程                   │
│                                                   │
│  涨停池/强势股 ──→ 拉取60日K线 ──→ 形态识别(7种)  │
│                                                   │

└─────────────────────────────────────────────────┘
```

## 二、七大K线信号详解

### C1 放量突破（权重 0.25）

**形态定义**：股价收盘价突破近20日最高价，且当日成交量 ≥ 5日均量的2倍

**市场含义**：多头力量集中爆发，突破前期压力位，开启新一轮上涨

**确认条件**：
- 收盘价 > 前20日最高价
- 量比 ≥ 2.0
- 当日阳线（收盘 > 开盘）

**操作参数**：

| 参数 | 数值 | 说明 |
|------|------|------|
| 止损 | -5% | 突破失败（跌回颈线下方） |
| 目标 | +15% | 突破后空间测算 |
| 仓位 | 100%基础仓位 | 满仓信号 |
| 持仓 | ≤10天 | |

**示例**：某股前20日最高价10.00元，今日收盘10.20元，量比2.5 → 放量突破信号

---

### C2 缩量回踩（权重 0.20）

**形态定义**：股价回踩10日均线附近（±2%），成交量萎缩至5日均量以下

**市场含义**：上涨趋势中的健康回调，主力洗盘结束，即将二次拉升

**确认条件**：
- 当日价格距10日均线 ≤ 2%
- 前4天中至少2天在均线上方（趋势向上）
- 当日成交量 < 5日均量

**操作参数**：

| 参数 | 数值 | 说明 |
|------|------|------|
| 止损 | -6% | 跌破均线且不收回 |
| 目标 | +12% | 回踩前的高点 |
| 仓位 | 90%基础仓位 | |
| 持仓 | ≤12天 | 回踩后需要时间发酵 |

---

### C3 倍量柱（权重 0.15）

**形态定义**：单日成交量 ≥ 5日均量2倍，且当日涨幅 > 3%，阳线收盘

**市场含义**：主力资金大举入场，是强烈的看多信号

**确认条件**：
- 涨幅 > 3%
- 量比 ≥ 2.0
- 阳线

**操作参数**：

| 参数 | 数值 | 说明 |
|------|------|------|
| 止损 | -7% | 倍量柱次日常有震荡 |
| 目标 | +15% | |
| 仓位 | 80%基础仓位 | |
| 持仓 | ≤8天 | |

**注意**：倍量柱次日需观察是否持续放量，若缩量下跌则信号减弱

---

### C4 早晨之星（权重 0.15）

**形态定义**：三根K线组合——大阴线 → 十字星 → 大阳线

**市场含义**：下跌趋势末端反转信号，空头力量衰竭，多头开始反攻

**确认条件**：
- 第1天：跌幅 > 2%的大阴线
- 第2天：十字星（实体/振幅 < 0.3，振幅 < 3%）
- 第3天：涨幅 > 2%的大阳线，收盘高于第1天实体中点

**操作参数**：

| 参数 | 数值 | 说明 |
|------|------|------|
| 止损 | -5% | 反转失败止损紧 |
| 目标 | +10% | |
| 仓位 | 70%基础仓位 | 反转信号轻仓 |
| 持仓 | ≤7天 | |

---

### C5 红三兵（权重 0.10）

**形态定义**：连续三根阳线，收盘价递增，每根涨幅 ≥ 1%，温和放量

**市场含义**：多头力量稳步增强，趋势确立

**确认条件**：
- 三根阳线（收盘 > 开盘）
- 收盘价递增
- 每根涨幅 ≥ 1%
- 第3天量 ≥ 第1天量 × 0.8（不大幅缩量）

**操作参数**：

| 参数 | 数值 | 说明 |
|------|------|------|
| 止损 | -5% | |
| 目标 | +12% | |
| 仓位 | 85%基础仓位 | |
| 持仓 | ≤10天 | |

---

### C6 锤头线（权重 0.10）

**形态定义**：下影线 ≥ 实体2倍，上影线 < 实体0.5倍，出现在下跌末端

**市场含义**：盘中空头打压被多头收复，底部支撑强劲

**确认条件**：
- 下影线 ≥ 实体 × 2
- 上影线 ≤ 实体 × 0.5
- 前5日累计跌幅 > 3%（下跌趋势中）

**操作参数**：

| 参数 | 数值 | 说明 |
|------|------|------|
| 止损 | -4% | 跌破锤头下影线最低点 |
| 目标 | +8% | 锤头线反弹空间有限 |
| 仓位 | 60%基础仓位 | 极轻仓抄底 |
| 持仓 | ≤5天 | 快进快出 |

---

### C7 阳包阴（权重 0.15）

**形态定义**：当日阳线完全包住昨日阴线，且成交量放大

**市场含义**：多头强力反吞，短线反转信号

**确认条件**：
- 昨日阴线（收盘 < 开盘）
- 今日阳线（收盘 > 开盘）
- 今日开盘 ≤ 昨日收盘，今日收盘 ≥ 昨日开盘
- 量比 ≥ 1.5

**操作参数**：

| 参数 | 数值 | 说明 |
|------|------|------|
| 止损 | -5% | |
| 目标 | +10% | |
| 仓位 | 75%基础仓位 | |
| 持仓 | ≤7天 | |

## 三、量能确认规则

量能是K线形态的"验证码"——没有量能配合的形态可靠性大打折扣。

### 量能三定律

```
┌─────────────────────────────────────────────┐
│  定律1：突破必须放量（量比≥2）                │
│  定律2：回踩必须缩量（量比<1）                │
│  定律3：反转必须见量（次日放量确认）           │
└─────────────────────────────────────────────┘
```

### 量能加成规则

| 条件 | 加成 | 说明 |
|------|------|------|
| 均线多头排列（MA5>MA10>MA20） | +10%仓位 | 趋势确认 |
| 近3日均量 > 前5日均量 × 1.2 | 评分+0.10 | 持续放量 |
| 多信号叠加（≥2个） | 仓位+5% | 信号共振 |
| 多信号叠加（≥3个） | 仓位再+5% | 强共振 |

## 四、均线系统

### 三条均线的作用

| 均线 | 作用 | 信号 |
|------|------|------|
| MA5 | 超短线止损参考 | 跌破MA5 → 减仓 |
| MA10 | 短线趋势线 | 回踩MA10企稳 → 低吸 |
| MA20 | 中线生命线 | 跌破MA20 → 清仓 |

### 均线排列判断

- **多头排列**：MA5 > MA10 > MA20 → 上升趋势，可操作
- **空头排列**：MA5 < MA10 < MA20 → 下降趋势，不参与
- **纠缠排列**：三线交叉缠绕 → 震荡市，轻仓或观望

|------|------|------|---------|---------|--------|
| 放量突破 | -5% | +15% | 1.00 | 10天 | 3.0 |
| 缩量回踩 | -6% | +12% | 0.90 | 12天 | 2.0 |
| 倍量柱 | -7% | +15% | 0.80 | 8天 | 2.1 |

> **盈亏比公式**：`盈亏比 = 目标涨幅 / |止损幅度|`
>
> 所有信号的盈亏比均 ≥ 2.0（风控最低要求），不达标时系统自动提高目标价。

## 六、与涨停连板策略的配合

两套策略独立运行，互为补充：

| 维度 | 涨停连板策略（路径A/B） | K线量能策略（路径C） |
|------|----------------------|---------------------|
| 选股范围 | 当日涨停池 | 涨停池+强势股 |
| 核心信号 | 连板数/封单/题材热度 | K线形态/量能配合 |
| 适合行情 | 强势行情（题材炒作） | 震荡市/趋势市 |
| 持仓周期 | 短线（1-5天） | 波段（5-12天） |
| 风险等级 | 高（追涨） | 中（形态确认） |

### 融合规则

1. **两策略同时命中**的股票 → 优先级最高，仓位可上浮10%
2. **仅K线信号命中**但不在涨停池 → 可独立操作，按K线策略参数执行
3. **仅涨停池命中**但无K线信号 → 按原涨停策略执行

## 七、风控规则（共用）

K线策略与涨停策略共用同一套风控规则（`screener/rules.json`）：

| 风控参数 | 数值 | 说明 |
|---------|------|------|
| 单只仓位上限 | 30% | 单只股票最多占总资金30% |
| 总仓位上限 | 80% | 永远留20%现金 |
| 日亏熔断 | -5% | 当日亏5%强制停手 |
| 单笔硬止损 | -8% | 单笔最大亏损8% |
| 最小盈亏比 | 2.0 | 目标/止损 ≥ 2 |
| 日最大交易 | 3笔 | 每天最多交易3次 |
| 最大持仓天数 | 10天 | 到期强制复盘 |
